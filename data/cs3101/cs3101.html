Section A: Multiple Choice Questions (10 Marks - 2 marks each)

  1. Which of the following is NOT a component of a microprocessor?

    a) ALU (Arithmetic Logic Unit)
    b) Control Unit
    c) Hard Disk Drive
    d) Registers

  2. What is the primary advantage of RISC architecture over CISC?

    a) More complex instructions
    b) Faster execution due to simpler instructions
    c) Larger instruction set
    d) Lower power consumption

  3. Which register stores the address of the next instruction to be executed?

    a) Accumulator
    b) Program Counter (PC)
    c) Stack Pointer
    d) Instruction Register

  4. What is the typical size of an instruction in an 8-bit microprocessor?

    a) 8 bits
    b) 16 bits
    c) 1-3 bytes
    d) Always 32 bits

  5. Which addressing mode uses the content of a register as an address?

    a) Immediate addressing
    b) Direct addressing
    c) Register indirect addressing
    d) Relative addressing
